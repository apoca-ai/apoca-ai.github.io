# Apoca AI 官网

Apoca AI 官方网站，基于 GitHub Pages 托管。

## 访问地址

https://apoca-ai.github.io

## 本地预览

```bash
python3 -m http.server 8080
```

然后访问 http://localhost:8080

## 技术栈

- HTML5
- CSS3 (自定义属性、Grid、Flexbox)
- JavaScript (原生，无框架依赖)

## 特性

- 响应式设计，适配各种设备
- 暗色主题设计
- 平滑滚动和交互动画
- 无外部依赖（除 Google Fonts 外）
